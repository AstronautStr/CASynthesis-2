# 2026-06-20 — Шаг B: формозависимые амплитуды (ручка `shape`)

## Что сделано

Реализован параметр `shape=0.0` в `map_laplacian` (`casynth_core.py`).

**Ключевые точки реализации:**
- `shape=0.0` → ветвь `eigvalsh` (бит-в-бит как раньше, без регрессии по скорости).
- `shape>0.0` → `eigh(L)`: собственные значения + векторы.
- Возбуждение `e = L.diagonal()` (graph-degree, граф-интринсик, инвариантен к позиции/повороту/отражению).
- Индексный массив ведётся сквозным через все три маски: `nonzero_idx → survived_idx[guard_mask] → survived_idx[sel]` → `final_idx` для столбцов `vecs`.
- Blend: `(1−shape)·rolloff + shape·proj`, normalize max=1.
- ENGINES: добавлен спек `('shape', 'shape', 0.0, 1.0, False, 0.0)`.

**Дизайн-вопрос по тесту C (P3-blinker):** Researcher подтвердил, что нулевая проекция e=[1,2,1] на антисимметричную λ=1-моду — корректное физическое поведение (симметричное возбуждение ≠ антисимметричная мода). Нули допустимы. Тест C сверяет точные значения `amps[:2]==[0.0,1.0]` — слабая проверка ("нет NaN + max=1") пропустила бы ошибку перестановки индексов.
