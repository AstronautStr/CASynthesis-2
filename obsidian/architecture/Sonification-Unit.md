# Единица сонификации = перцептивный объект

## Решение
**Не клетка, не всё поле — каждый связный объект поля.**

Поле = аудиальная сцена (Bregman ASA / Gestalt grouping). Каждый объект → один голос/партиал.

## Следствия
- Связность — 8-connectivity (диагонали считаются)
- Голосовой кап: MAX_VOICES = 24, приоритет крупным объектам
- Объект исчезает → голос умолкает; появляется → стартует

## Открытые технические проблемы
- Сегментация не wrap-aware (объекты на тороидальном шве временно раздваиваются)
- Нет identity tracking между поколениями (нет настоящих birth/death/merge/split событий)
