# Стратегия: большое исследование маппингов

## Принцип
**Не выбрасывать ветки — профилировать.** Законсервированный приём — не отвергнутый. Разные приёмы могут управлять разными параметрами синта.

## Что фиксировать по каждому приёму

| Поле | Описание |
|---|---|
| **Характер** | Что слышно на слух («металлический», «шумовой», «цифровой») |
| **Разделение малые/крупные** | Насколько разные формы дают разный звук |
| **Роль в синте** | Тембр / огибающая / модуляция / пространство |
| **Чувствительность к режиму** | Малые осцилляторы / хаос / рост / стабилизация |
| **Инварианты** | К чему нечувствителен (позиция/поворот/отражение) |
| **Статус** | Активная / законсервирована / отвергнута + почему + на каких данных |

## Текущие профили

- [[Laplacian]] — активная, металлический тон, малые формы ✅
- [[FFT-2D]] — законсервирована, малые формы невыразительны, крупные не проверены
- [[Walsh-Hadamard]] — законсервирована, аналог FFT
- [[Random-Baseline]] — baseline, не кандидат
- [[Granulo]] — законсервирована, вырождена на тонких (диагноз), крупные не проверены

## Ближайшая задача
Прогнать законсервированные на [[Large-Dense-Fields]] — проверить гипотезу «шумный на малом → структурный на большом».

## Карта «какой приём для чего»
Итог исследования — распределение ролей:
- Один приём → тембр/спектр
- Другой → огибающая/динамика
- Третий → модуляция/детюн
- Один приём может попасть в несколько ролей
