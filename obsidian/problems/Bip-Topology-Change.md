# Бипы при смене топологии (Laplace)

#solved

## Симптом
На Beacon и Toad при смене шага GOL — резкий «бип». Появляется точка в центре формы → топология меняется → набор резонансных мод Лапласиана меняется скачком → новые частоты входят мгновенно на полной амплитуде с phase-reset → ступенька + разрыв наклона волны.

## Диагноз
- **Дизайн (сохранить):** смена набора частот при смене топологии — суть Laplace. Другая форма = другой резонатор = другие моды.
- **Артефакт (лечить):** КАК новые моды входят — мгновенно, полная амплитуда, phase-reset в 0.

## Решение (принято, 2026-06-15)
**Короткий амплитудный кросс-фейд наборов мод:**
- Исчезающие/меняющие частоту слоты → fade-out (глайд amp → 0)
- Новые слоты → fade-in из amp=0
- Phase-reset слота — только когда его amp = 0
- Длительность: ~40–60 мс (FADE_CHUNKS чанков), не зависит от темпа GOL
- Частоты мод НЕ глайдятся (дискретны)

Физическая аналогия: старая пластина затухает, новая вступает.

## Критерий приёмки
На Beacon/Toad при смене шага GOL — смена резонансов как мягкое событие (атака/затухание), без резкого «бипа»; частоты точные, дискретность сохранена; низшая мода держит f0.
