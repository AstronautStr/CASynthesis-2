# Granulo (гранулометрия — распределение масштабов структуры)

#conserved

## Алгоритм
Серия `scipy.ndimage.binary_opening` с возрастающим радиусом структурного элемента → распределение «сколько структуры выжило при стирании масштаба r». Мелкий масштаб → высокие гармоники (после реверса оси).

## Перцептивный характер
На мелком зоопарке — **вырожден**: один высокий синус для всех форм. На своём материале (крупные плотные поля) — «масштаб деталей → яркость».

## Разделяющая способность
- Малые формы: **идентичны** — нет разрешения на тонких формах (3–8 клеток). Все четыре осциллятора дают один и тот же вектор — диагноз подтверждён прослушиванием
- Крупные/плотные: **не прослушано** — именно там у структуры есть масштабы → ожидаемо информативен

## Диагноз (почему вырожден)
Минимальный структурный элемент — диск радиуса 1 (крест из 5 клеток). Все тонкие осцилляторы (толщина 1–2 клетки) стираются на первой же итерации → вся масса в одном бине → один и тот же партиал для всех форм.

**Это «не тот материал», а не плохой приём.**

## Инварианты
Трансляция, поворот (морфологические операции).
